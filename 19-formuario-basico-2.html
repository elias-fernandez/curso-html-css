<label>

El Elemento HTML <label> representa una etiqueta para un elemento en una interfaz de usuario

Este puede estar asociado con un control ya sea mediante la utilizacion del atributo for, o ubicando el control dentro del elemento label. Tal control es llamado "el control etiquetado" del elemento label.

Atributos <label>

  • for

    El ID del elemento de formulario etiquetable relacionado en el mismo documento que el elemento label.

    En buestro caso observa que el id del primer input es "hombre" y ese mismo valor es el que se va a utilizar para el atributo for en label

    Podriamos decir: crear una etiqueta para el id=XYZ

    El primer elemento en el documento con tal ID que coincida con el dispuesto en el atributo for será el control etiquetado para este elemento.

Atributos <input>

  • id:

    Identificador único

  • type="radio":

    Tipo de control a mostrar, en ste caso un radio button (para elegir una sola opción)

  • name="sexo":

    Nombre para agrupar los distintos botones del formulario que pertenecen a un mismo gupo/categoria/conjunto

    Observa en nuestro ejemplo que nuestros 3 botones poseen este atributo identicamente, reflejando así que cualquiera de estos inputs pertenecen a una categoria
