Colores en CSS

Podemos referenciar el color mediante su nombre, un código hexadecimal o código rgb, cómo se explica a continuación

  • color:

    red, blue, green

  • Código Hexadecimal: #rrggbb o #rgb

    Valores validos: 0-9 A-F

    Si los valores se repiten se puede abreviar #22aaff = #2af

  • Código rgb + a (opcional)

    Valores válidos para rgb: 0...255

    Valores válidos para a (alpha o canal de transparencia): 0...1, siendo 0 totalmente transparente y 1 totalmente solido, acepta valores intermedios

    rgb(255, 77, 255)

    rgba(255, 77, 255, 0.75)
