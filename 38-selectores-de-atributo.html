Selectores de atributos en CSS

  1. Selector de atributo simple [att]: Este selector selecciona elementos que tienen un atributo específico, independientemente de su valor. En este caso, se aplicará un estilo a todos los enlaces <a> que tengan el atributo target, sin importar cuál sea su valor.Por ejemplo:

    /* Selecciona todos los elementos con el atributo "target" */

          a[target] {
            color: blue;
          }

  2. Selector de atributo [att=val]: Este selector selecciona elementos que tienen un atributo con un valor específico. Este estilo se aplicará solo a los enlaces que tengan el atributo target con el valor "_blank". Por ejemplo: 

    /* Selecciona todos los enlaces que tengan un atributo "target" con el valor "_blank" */
          
          a[target="_blank"] {
            text-decoration: underline;
          }

  3. Selector de atributo de palabra clave [att~=val]: Este selector selecciona elementos cuyo atributo tiene un valor que contiene una palabra clave específica, separada por espacios. Este estilo se aplicará a cualquier elemento que tenga un atributo class con la palabra clave "important", independientemente de las otras clases que pueda tener. Por ejemplo:

    /* Selecciona elementos que tengan un atributo "class" que contenga la palabra clave "important" */
          
          .important {
            font-weight: bold;
          }

  4. Selector de atributo que comienza con [att|=val]: Este selector selecciona elementos cuyo atributo comienza con un valor específico seguido de un guion (-) o está igualado a ese valor. Es útil principalmente para seleccionar elementos con atributos de idioma, como el atributo lang. Este estilo se aplicará a elementos con el atributo lang igualado a "es" o que comience con "es-". Por ejemplo:

    /* Selecciona elementos con el atributo "lang" igualado a "es" o que comienza con "es-" */

          [lang|=es] {
          font-family: Arial, sans-serif;
          }
